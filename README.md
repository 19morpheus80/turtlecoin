### How To Compile

#### Ubuntu 16.xx LTS
- `sudo apt-get update`
- `sudo apt-get -y install build-essential python-dev gcc-4.9 g++-4.9 git cmake libboost1.58-all-dev librocksdb-dev`
- `export CXXFLAGS="-std=gnu++11"`
- `git clone https://github.com/turtlecoin/turtlecoin`
- `cd turtlecoin`
- `mkdir build && cd $_`
- `cmake ..`
- `make`
